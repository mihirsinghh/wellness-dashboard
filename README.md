# Wellness Dashboard

A Vite + React wellness dashboard for habits, tasks, expenses, and journaling.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The production build outputs to `dist/`.

## Deploy on Vercel

1. Push this repository to GitHub.
2. Import the repo into Vercel.
3. Vercel should detect the app as a Vite project automatically.
4. Confirm these settings if prompted:

- Build Command: `npm run build`
- Output Directory: `dist`

No server or environment variables are required for the current app.
